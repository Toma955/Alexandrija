# Media

Mapa medija za Alexandria.

## Struktura

- **sounds/** – zvukovi (npr. sistemski zvučni efekti, notifikacije)
- **images/** – slike (pozadine, ilustracije, zasloni)
- **icons/** – ikone
  - **classic/** – klasični set ikona
  - **default/** – zadani set ikona

Dodaj datoteke u odgovarajuće podmape; Xcode ih uključuje u build putem File System Synchronized Group.
